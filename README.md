# 🎲 Dice Roll Game (Python)

## 📌 Project Overview

This is a simple Dice Roll Game developed using Python. The program simulates rolling a dice for two players and declares the winner based on the result.

The project is menu-driven and designed to demonstrate basic programming concepts like functions, loops, and random number generation.

---

## 🚀 Features

* 🎲 Random dice rolling using Python's random module
* 👥 Two-player gameplay
* 📋 Menu-driven interface
* 🏆 Winner announcement after each round
* ❌ Input validation for better user experience

---

## ▶️ How to Run the Project

1. Install Python on your system
2. Download or clone this repository
3. Open the project folder
4. Run the file:

```
python dice_game.py
```

---

## 🎮 How to Play

1. Select "Play Game" from the menu
2. The system rolls dice for both players
3. The player with the higher number wins
4. In case of equal numbers, it’s a tie
